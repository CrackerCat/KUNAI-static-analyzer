Detailed Description

Class used to manage all the information from the analysis it contains the xrefs between Classes, Methods, Fields and Strings. Also it creates the BasicBlocks from the code. XREFs are created for:

Analysis definition

  • classes (ClassAnalysis object)
  • methods (MethodAnalysis object)
  • strings (StringAnalysis object)
  • fields (FieldAnalysis object)
